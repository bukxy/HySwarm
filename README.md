# HySwarm

A Hystale mod inspirated by [MegaBonk](https://store.steampowered.com/app/3405340/Megabonk/).

### Fonctionalities (WIP)

- ✅ 360° orbital camera 
- ⚒️ Wave system — one wave every 10 minutes, up to 3 maps, followed by a final boss map
- ⚒️ Mini bosses
- ⚒️ Custom items
- ⚒️ Special waves
- ⚒️ Quests
